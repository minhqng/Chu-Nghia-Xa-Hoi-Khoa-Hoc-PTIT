# CNXH PTIT Quiz

Web app on luyen trac nghiem mon Chu Nghia Xa Hoi Khoa Hoc cho PTIT.

## Tinh nang

- Chon nguon hoc: `Ngân Hàng Câu Hỏi` hoac `Ôn Tập Lý Thuyết`.
- Chon tung chuong hoac tat ca chuong trong che do on tap.
- Che do `Ôn tập`: hien dung/sai ngay, hien giai thich keyword, tu sang cau sau neu tra loi dung.
- Che do `Thi thử`: boc ngau nhien tu ca 2 thu muc, 1 phut/cau, chi hien ket qua sau khi nop bai.
- Ma tran cau hoi, progress bar, timer, bo dem dung/sai/tong.
- Tu luu tien do bang localStorage va co nut lam lai.

## Lenh chay

```bash
npm install
npm run dev
npm run validate:data
npm run test
npm run build
```

## Du lieu

Ung dung import truc tiep 14 file JSON trong 2 thu muc du lieu goc. Chay
`npm run validate:data` sau khi sua JSON de bat loi cau trong, sai so dap an dung,
hoac so option khong hop le.
